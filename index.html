Política de Privacidade - Sylph

A Sylph valoriza sua privacidade. Os dados de saúde (peso, altura e caminhadas) são fornecidos manualmente pelo usuário e usados apenas para cálculos internos.

As fotos enviadas no chat são redimensionadas localmente para economizar memória e processadas apenas para interação com a IA.

Não compartilhamos seus dados com terceiros. Para dúvidas ou exclusão de dados, entre em contato: Sylph.app@gmail.com
